## How to run?

Rails Command:

- Install package: bundle install
- Migrating database: rails db:migrate
- Seeding database: rake db:seed
- Run server: rails s -e dev
  \n
  Tailwindcss Command:
- Tailwind dev mode: rails tailwindcss:watch
- Tailwind buid: rails tailwindcss:build
